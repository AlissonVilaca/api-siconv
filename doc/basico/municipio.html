Métodos

Consultar os dados de municípios.

Município

Método:

Consultar Municípios

Descrição:

Fornece uma lista com informações relacionadas a municipios dentro do território brasileiro. Cada município tem sua própria URI e ID que compõe a mesma.

URL:

http://api.convenios.gov.br/siconv/v1/consulta/municipios.<formato>

Formatos:

xml, json, cvs, html

Detalhar:

Documentação sobre detalhamento do município.

Exemplo:

http://api.convenios.gov.br/siconv/v1/consulta/municipios.html?uf=PE
