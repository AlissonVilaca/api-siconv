Esfera Administrativa

Método:

Detalhar Esfera Administrativa

Descrição:

Informações detalhadas.

URL:

http://api.convenios.gov.br/siconv/dados/esfera_administrativa/<ID>.<formato>

Formatos:

rdf, xml, json, cvs, n3

Exemplo:

http://api.convenios.gov.br/siconv/dados/esfera_administrativa/1.xml

Parâmetros da Solicitação

Esse método é utilizado para consultar dados de uma única esfera administrativa, além do ID fornecido na composição da URL não há parâmetros para filtrar o resultado.
