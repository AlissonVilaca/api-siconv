Proposta

Detalhar os dados da proposta.

Proposta

Descrição:

Informações detalhadas.

URL:

http://api.convenios.gov.br/siconv/dados/proposta/<ID>.<formato>

Formatos:

rdf, xml, json, cvs, n3

Exemplo:

http://api.convenios.gov.br/siconv/dados/proposta/379143.xml

Parâmetros da Solicitação

Esse método é utilizado para consultar dados de uma única proposta, além do ID fornecido na composição da URL não há parâmetros para filtrar o resultado.
