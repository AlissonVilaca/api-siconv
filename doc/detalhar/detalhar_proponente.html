Proponente

Detalhar os dados do proponente.

Proponente

Descrição:

Informações detalhadas.

URL:

http://api.convenios.gov.br/siconv/dados/proponente/<ID>.<formato>

Formatos:

rdf, xml, json, cvs, n3

Exemplo:

http://api.convenios.gov.br/siconv/dados/proponente/32306.xml

Parâmetros da Solicitação

Esse método é utilizado para consultar dados de um único proponente, além do ID fornecido na composição da URL não há parâmetros para filtrar o resultado.
