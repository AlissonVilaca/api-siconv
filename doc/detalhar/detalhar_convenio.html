Métodos

Detalhar os dados do convênio.

Convênio

Método:

Detalhar Convênio

Descrição:

Informações detalhadas.

URL:

http://api.convenios.gov.br/siconv/dados/convenio/<ID>.<formato>

Formatos:

rdf, xml, json, cvs, n3

Exemplo:

http://api.convenios.gov.br/siconv/dados/convenio/1702.xml

Parâmetros da Solicitação

Esse método é utilizado para consultar dados de um único convênio, além do ID fornecido na composição da URL não há parâmetros para filtrar o resultado.
