Métodos

Consultar o cadastro dos convênios.

Convênios

Método:

Consultar por Convênios

Descrição:

Fornece uma lista com informações relativas a convênios firmados com o Poder Executivo Federal. Cada convênio tem sua própria URI e ID que o compõe.

URL:

http://api.convenios.gov.br/siconv/v1/consulta/convenios.<formato>

Formatos:

xml, json, cvs, html

Detalhar:

Documentação sobre detalhamento do convênio.

Exemplo:

http://api.convenios.gov.br/siconv/v1/consulta/convenios.html?uf=RS
